QA Form Tool

Installation

To install this tool, simply drag the green button below to your browser's Bookmarks Bar.

QA Hello World

How to use

  1. Go to any website (e.g., Google, your company site, etc.).
  2. Click the QA Hello World bookmark you just created.
  3. The floating window should appear in the bottom right corner.
Note: Since this version is self-contained, if you ever update the code in the future, you will need to come back to this page and re-drag the button to update your bookmark.
